# 📊 Smart Stock Insight Platform (AI-Based)

A modern **AI-powered stock analysis and financial literacy platform** that provides real-time stock data, trend prediction, and interactive insights using simple yet powerful algorithms.

---

## 🚀 Project Overview

This project is built to combine **financial literacy + coding + AI concepts** into a single interactive dashboard.

It allows users to:

* View real-time stock data 📈
* Understand financial concepts 📚
* Analyze trends using AI-based logic 🤖
* Visualize predictions on graphs 📊

---

## 🧠 Key Features

### 📈 1. Live Stock Dashboard

* Fetches real-time stock data using Yahoo Finance
* Displays:

  * Current Price
  * Day High / Low
  * Market Cap
  * Price Change (%)

---

### 🤖 2. AI-Based Prediction System

Advanced **multi-factor analysis engine** including:

* RSI (Relative Strength Index)
* SMA (Moving Average Crossover)
* Volume Trend Analysis
* Volatility Detection
* Momentum (3-day trend)

➡️ Generates:

* AI Signal → Buy / Sell / Hold
* Confidence Score (%)
* Predicted Next Price 🔮

---

### 📊 3. Graph-Based Prediction

* Interactive chart using Chart.js
* Shows:

  * Historical prices
  * Predicted future trend (dotted line)
* Visual understanding of stock movement

---

### 📉 4. Advanced Indicators

* Trend Strength Meter (Weak / Moderate / Strong)
* Risk Level Indicator (Low / Medium / High)
* Support & Resistance Levels
* AI Score Meter (-10 to +10)

---

### 💡 5. Smart AI Insights

System generates human-readable explanations like:

> “Stock shows bullish momentum with increasing volume and moderate risk.”

---

### 📚 6. Financial Literacy Section

Beginner-friendly explanations of:

* Stock
* Share
* Investment
* Profit
* Risk
* Dividend
* P/E Ratio
* Diversification

---

## 🛠️ Tech Stack

### Frontend:

* HTML
* CSS (Modern Glass UI)
* JavaScript
* Chart.js

### Backend:

* Python (Flask)
* yFinance API

---

## ⚙️ How It Works

1. User selects a stock
2. Backend fetches real-time data
3. AI engine analyzes:

   * Price trends
   * Indicators
4. System generates:

   * Prediction
   * AI signal
   * Graph visualization

---

## ▶️ How to Run

### 1. Install dependencies

```bash
pip install flask yfinance
```

### 2. Run server

```bash
python server.py
```

### 3. Open browser

```
http://127.0.0.1:5000
```
